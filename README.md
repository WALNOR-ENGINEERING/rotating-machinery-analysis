# Rotating Machinery Analysis

![Cover](cover.png)

Engineering studies focused on rotating systems,
aerodynamic behavior and flow interaction
inside dynamic machinery.

The objective is not only to visualize rotating flow behavior,
but to understand aerodynamic interaction,
pressure generation and system efficiency.
